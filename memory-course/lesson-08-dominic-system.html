üìö The Dominic System Explained

üéØ What is the Dominic System?

Created by 8-time World Memory Champion Dominic O'Brien, this system converts number pairs into PEOPLE and ACTIONS.

  • Each number pair (00-99) = one specific person
  • You assign each person an action
  • Much easier to remember famous people than abstract words!
  • Great for competing in memory championships

Key Advantage: Famous people are MUCH more memorable than random words!

üîç Dominic vs Major System

üìä Head-to-Head Comparison

üîê Major System

  • Uses consonant words
  • Must create vowels
  • More universal
  • Words sometimes random

üë§ Dominic System

  • Uses famous people
  • Assigns actions
  • More memorable
  • Personal & emotional

üéØ When to Use Each System

Use Major System when:

  • You need quick, flexible word creation
  • The language isn't English
  • You're just learning memory

Use Dominic System when:

  • You want MAXIMUM memorability
  • You have time to build a personal list
  • You're competing in memory championships
