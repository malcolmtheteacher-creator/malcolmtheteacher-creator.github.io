LESSON 14: MIND MAPPING FOR MEMORY
File: lesson-14-mind-mapping.html (25 KB)
================================================================================

Content:
- Tony Buzan's Mind Mapping Rules:
  * Central Image: Bold, vivid, colorful main concept
  * Main Branches: 5-7 primary branches with keywords
  * Curved Lines: Organic connections mimicking brain structure
  * Colors & Symbols: Visual encoding through colors/icons
  * Keywords Only: Single words per line for associations
- Mind map to palace conversion
- Hierarchical structure (levels 1-4)
- Dual encoding benefits (visual + spatial + semantic)

Interactive Features:
- Interactive mind map builder with branch generator
- Dynamic branch and sub-branch input creation
- Visual mind map preview with card-based display
- Centroid node visualization
- Palace conversion workflow

JavaScript Implementation:
- Nested array structure for hierarchical data
- Dynamic DOM element creation for branches
- Branch management (add/edit sub-branches)
- Visual preview generation
- Interactive input field handling
