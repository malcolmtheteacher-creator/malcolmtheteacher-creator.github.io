Recall Zone — From Previous Lessons

Let's see what you remember from before. Try from memory — no looking!

Recall 1: Greetings & Introductions

How many ways can you say hello and introduce yourself? Write at least 3 different ways.

Some options: "Hello, my name is..." / "Hi, I'm..." / "Nice to meet you." / "How are you?" / "I'm fine, thank you."

Recall 2: Common Objects

Write 5 objects you can find in a classroom. From memory!

Possible answers: desk, chair, board, book, pen, pencil, bag, computer, window, door, clock, ruler, eraser

Recall 3: "There is / There are"

Complete from memory: When do we use "there is" and when do we use "there are"?

There is + singular (one thing): "There is a book on the table."
There are + plural (two or more): "There are three chairs in the room."
