How to get a token (once per device)

1. GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new (classic).

2. Name it "paracosm". Under scopes tick only gist. Generate, copy it, paste it above.

3. Click Connect & sync — a private gist is created to hold your sections.

4. On another device: open this page, paste the same token and the Gist ID shown here, then Connect.
